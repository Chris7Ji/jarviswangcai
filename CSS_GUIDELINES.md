# CSS 颜色对比度规范 (Color Contrast Guidelines)

## 核心原则

**颜色必须跟随上下文背景自动适配**，而不是机械套用某个固定颜色。根据目标容器的背景颜色选择对应的文字/链接颜色。

## 颜色决策流程（自动化脚本必须遵循）

```
生成内容颜色时，按以下顺序判断：

1. 我要放的这个元素/文字，落在哪种背景上？
   ├── 浅色背景（#FEFBF6 / #FFFFFF / #F5F0E8）
   │   → 文字用：#2C3E50（深蓝灰）
   │   → 链接用：var(--tech-blue) #3D9CA8
   │   → 禁用：任何 rgba(255,255,255,*) 的文字
   │
   ├── 深色背景（#2C3E50 / #1a252f / 深色自定义）
   │   → 文字用：rgba(255,255,255,0.85~0.95)（非纯白）
   │   → 链接用：#7FDBFF（浅蓝）或 var(--primary-gold) #D4A84B
   │   → 禁用：#FFFFFF（纯白对比度过高刺眼）
   │
   └── 卡片/区块（白色背景 #FFFFFF）
       → 文字用：#2C3E50
       → 链接用：var(--tech-blue)
```

---

## 关键禁区

### ❌ 绝对禁止
- 在**任何浅色背景**上使用 `color: white`、`#fff`、`rgba(255,255,255,*)`
- 在**任何深色背景**上使用纯白 `#FFFFFF`（对比度过高，应使用 `rgba(255,255,255,0.85~0.95)`）
- 在文章页 `.post-content` 内直接写 `style="color: white"`
- 在 `.navbar`（深色背景）上写 `color: #000000`
- 链接颜色与正文颜色完全相同

### ✅ 正确示范
```html
<!-- 浅色背景的文章页 -->
<p style="color: #2C3E50;">
  正文文字，<a href="#" style="color: #3D9CA8;">这是链接（tech-blue）</a>
</p>

<!-- 深色背景的文章延伸阅读区 -->
<div style="background: #2C3E50; color: rgba(255,255,255,0.85);">
  <a href="#" style="color: #7FDBFF;">链接用浅蓝</a>
</div>
```

## 自动化脚本要求

所有生成 HTML 的脚本必须：

1. **判断目标元素的父容器背景色**，选择对应配色
2. **优先使用 CSS 类**，避免内联 `style="color:..."`
3. **禁止硬编码白色系**（`#fff`、`white`、`rgba(255,255,255,*)`）到浅色背景容器
4. **禁止硬编码深色文字**到深色背景容器
5. **链接必须与正文颜色不同**（使用上述配色对照表）

---

## 快速检查清单（每次提交前）

- [ ] 新增/修改的文字颜色，是否与所在背景对比度达标（≥ 4.5:1）？
- [ ] 链接颜色是否与同区域普通文字颜色不同？
- [ ] 浅色背景区域是否有白色/浅色文字？（禁止）
- [ ] 深色背景区域是否用了纯白 `#FFFFFF`？（应改为 rgba）
- [ ] 深色背景区域是否用了深色文字？（禁止）

---

## 背景不是白色/米色时的通用规则

如果遇到**自定义背景色**（非上述已知颜色），使用以下公式判断：

```
文字颜色 = 取背景色的反色 或 低饱和度版本
链接颜色 = 背景色的互补色（+15°~30°色相偏移）
```

**简单判断法**：如果背景深，文字用 rgba(255,255,255,0.85~0.95)；如果背景浅，文字用 #2C3E50。
