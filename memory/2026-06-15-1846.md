## 🚀 配置阿里百炼模型

### 1️⃣ 获取 API Key

登录 [bailian.console.aliyun.com](https://bailian.console.aliyun.com) → API-KEY 管理 → 创建 API Key

有两种计费方式：
- **按量付费（Standard）**：用多少付多少，推荐
- **订阅计划（Coding Plan）**：包月订阅

### 2️⃣ 在 OpenClaw 中配置

**方式一：一键 onboarding（推荐）**

```bash
# 按量付费（国内端点）
openclaw onboard --auth-choice qwen-standard-api-key-cn

# 或者 Coding Plan（国内端点）
openclaw onboard --auth-choice qwen-api-key-cn
```

**方式二：设环境变量**

```bash
export QWEN_API_KEY=***
```

（同时兼容 `DASHSCOPE_API_KEY` 和 `MODELSTUDIO_API_KEY`）

### 3️⃣ 验证配置

```bash
openclaw models list --provider qwen
```

### 4️⃣ 设为默认模型（可选）

配置后，你可以在 `openclaw.json` 里把某个百炼模型设为主力：

```json5
{
  agents: {
    defaults: {
      model: { primary: "qwen/qwen3.5-plus" },
    },
  },
}
```
